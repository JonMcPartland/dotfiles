# My Dotfiles

## Dependencies & Assumptions
- OS X  
- Homebrew  
- Bash v4  
- iTerm 2 (xterm-256)  
- rbenv  
- [thefuck](https://github.com/nvbn/thefuck)  


## Installation
Clone into `~/.dotfiles/`:  
```bash
cd ~ && git clone https://github.com/JonMcPartland/dotfiles.git .dotfiles
```

Symlink to `.bash_profile`:  
```bash
ln -s ~/.dotfiles/bash-profile ~/.bash_profile
```
